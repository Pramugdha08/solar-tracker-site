OUR PROJECT

A) Introduction

The solar tracker project originated from the desire to optimize solar energy harvesting by aligning solar panels dynamically with the sun’s position throughout the day. Traditional fixed solar panels capture only a fraction of the available sunlight, limiting overall energy efficiency. The core theme of this project is to design an intelligent, automated solar tracking system using low-cost components that can enhance power generation without requiring expensive or complex machinery.


By employing microcontrollers, light-dependent resistors (LDRs), and servo motors, the project aims to build a compact, efficient solution that continuously adjusts the orientation of a solar panel to maximize incident sunlight. This real-time tracking maximizes output energy, making solar power systems more viable for residential and small-scale commercial use. Integration with an ESP01-12E Wi-Fi module further enables remote monitoring via a web interface, demonstrating the incorporation of IoT principles.


This project not only exemplifies the intersection of renewable energy and embedded systems but also serves as a practical learning platform for electronics, programming, and sustainable engineering.

 LDR1 connected to Analog Pin A0 (via voltage divider)

 LDR2 connected to Analog Pin A1 (via voltage divider)

 Servo Motor connected to Digital PWM Pin 9


o Display

 LCD connected via I2C pins (SDA to A4, SCL to A5) or parallel pins depending on model


o Communication

 ESP01-12E connected via Software Serial (e.g., Pins 2 and 3 for RX and TX)

 Ensure logic level shifting for 3.3V compatibility with ESP01 module


The solar panel is mounted on a mechanical platform attached to the servo motor shaft. The sensor inputs from the LDRs inform the Arduino about light intensity differences, which it processes to adjust the servo motor’s angle to face the direction of maximum sunlight.


D) Operation and Working of the Project

Solar Panel: The 12V solar panel captures sunlight and converts it into electrical energy. Its efficiency depends largely on the angle relative to the sun’s rays.


LDR Modules: The two LDR sensors detect light intensity from two different sides of the solar panel. Each LDR’s resistance changes inversely with light intensity. The Arduino reads analog voltage values from these sensors, allowing it to compare which side receives more sunlight.


Arduino Uno: The microcontroller acts as the project’s brain. It continuously samples analog readings from the LDRs, computes the difference, and decides if the servo motor needs to rotate to better align the solar panel towards the sun. It also manages communication with the LCD for local display and the ESP01 for web-based monitoring.


Servo Motor: The servo motor receives PWM signals from the Arduino to rotate the solar panel platform. Its precision enables smooth and incremental adjustments to optimize solar exposure.


LCD Display: The LCD shows real-time data including sensor values, motor angle, and system status, providing local feedback to the user.


ESP01-12E Wi-Fi Module: This module connects the system to a local Wi-Fi network, allowing real-time data transmission to a website or dashboard. This enables remote monitoring of solar panel orientation and sunlight intensity, enhancing usability and maintenance. Together, these components form a closed-loop control system that ensures the solar panel remains oriented toward the sun throughout the day, increasing total energy harvested compared to a static panel.


E) Arduino Code

#include


// Pins

const int LDR1 = A0;

const int LDR2 = A1;

const int servoPin = 9;

const int espRx = 2; // Arduino RX (to ESP TX)

const int espTx = 3; // Arduino TX (to ESP RX)


Servo solarServo;

LiquidCrystal_I2C lcd(0x27, 16, 2);

SoftwareSerial espSerial(espRx, espTx);


int servoPos = 90; // Start at mid-position


void setup() {

  Serial.begin(9600);

  espSerial.begin(115200);

  solarServo.attach(servoPin);

  solarServo.write(servoPos);

  lcd.init();

  lcd.backlight();

  lcd.setCursor(0,0);

  lcd.print("Solar Tracker");

  delay(2000);

}

void loop() {

  int ldrValue1 = analogRead(LDR1);

  int ldrValue2 = analogRead(LDR2);

  int error = ldrValue1 - ldrValue2;


  if(abs(error) > 20) { // Threshold to avoid jitter

    if(error > 0 && servoPos < 180) {

     servoPos += 1;

    } else if(error < 0 && servoPos > 0) {

    servoPos -= 1;

   }

   solarServo.write(servoPos);

  }


  // Display readings

  lcd.clear();

  lcd.setCursor(0,0);

  lcd.print("LDR1:");

  lcd.print(ldrValue1);

  lcd.setCursor(0,1);

  lcd.print("LDR2:");

  lcd.print(ldrValue2);


  // Send data to ESP for web monitoring

  espSerial.print("LDR1:");

  espSerial.print(ldrValue1);

  espSerial.print(",LDR2:");

  espSerial.print(ldrValue2);

  espSerial.print(",ServoPos:");

  espSerial.println(servoPos);


  delay(500);

}


Note: This code is a simplified version. Actual implementations may include error handling, Wi-Fi connection management, and enhanced smoothing algorithms.

F) Future Scopes of the Project

• Use of Two Servo Motors: Future enhancements can include two-axis tracking using two servo motors—one for horizontal (azimuth) and one for vertical (elevation) movement—allowing the solar panel to follow the sun’s path more precisely throughout the year.


• Advanced ESP01 Usage: Upgrading the ESP01 to support MQTT or HTTP protocols can improve integration with IoT platforms. Implementing cloud connectivity allows users to analyze historical data, predict energy output, and receive maintenance alerts via mobile apps.


• Energy Storage Integration: Combining the tracker with battery systems and smart charge controllers will enable autonomous solar energy storage, making the system useful during nighttime or cloudy weather.


• Machine Learning Algorithms: Incorporating ML to predict sun paths and optimize tracking based on weather forecasts and geographical data can increase efficiency and reduce mechanical wear.


• Scalability:The design can be scaled to larger panels and industrial-grade components, making it viable for commercial solar farms.
